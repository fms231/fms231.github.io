集合

主要分为两组：单列集合、双列集合

Collection接口和常用方法
add:添加一个元素
remove:移除元素
contains:是否包含某个元素
size:集合大小
isEmpty:是否为空
clear:清空集合
addAll:添加多个元素
containsAll:是否包含多个元素
removeAll:移除多个元素

遍历方式

  • 迭代器
  • 增强for循环
  • lambda表达式：forEach
